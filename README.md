# HMM_example_with_Dynamax

This notebook runs through an example of how to fit a generalised linear model-hidden Markov model (GLM-HMM) using the [Dynamax](https://probml.github.io/dynamax/index.html) package, following the protocol in [Ashwood et al., 2021](https://www.nature.com/articles/s41593-021-01007-z).

Data can be downloaded from [The Confidence Database](https://osf.io/s46pr/overview). The dataset used in the example is from [Rausch & Zehetleitner, 2016](https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2016.00591/full). The dataset should be placed in a 'data/' folder within the repo.
